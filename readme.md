# MouseJumpTool

A Windows tool that helps you quickly switch your mouse cursor between multiple monitors.

## Features

- **Automatic Multi-Monitor Detection**: Automatically detects all connected monitors in your system
- **Countdown Timer**: Displays a countdown timer on the target monitor, prompting you to move your mouse
- **Cursor Indicator**: Shows your cursor position with support for both rounded rectangle and circle styles
- **Single Instance**: Ensures only one instance runs at a time
- **System Tray Icon**: Supports start and exit operations via the system tray icon

## How to Use

1. Run the program - it will automatically detect and display all connected monitors
2. Move your mouse to the target monitor edge - a countdown will appear on the next monitor
3. After the countdown ends, the cursor will jump to the target monitor
4. Use the system tray icon to exit the program

## System Requirements

- Windows 7 or later
- Support for multi-monitor configuration

## Building

Use Visual Studio to open the `MyMouneTools.sln` solution file for compilation.

## 機能特性

- **自動検出マルチモニター**: システム内のすべての接続されているモニターを自動的に検出します
- **タイマー表示**: 次のモニターにカウントダウンタイマーを表示し、マウスを動かすことを促します
- **カーソル表示**: 丸角矩形と円形の両方のスタイルをサポートしています
- **シングルインスタンス**: 同時に 1 つのみが実行されます
- **システムトレイアイコン**: ステムトレイアイコンから開始と終了操作をサポートします

## 使用方法

1. プログラムを実行すると、自動的に接続されているすべてのモニターが検出されて表示されます
2. ターゲットモニターの端にマウスを移動させると、次のモニターにカウントダウンが表示されます
3. カウントダウンが終了すると、カーソルはターゲットモニターにジャンプします
4. システムトレイアイコンを使用してプログラムを終了します

## システム要件

- Windows 7 以降
- マルチモニター構成をサポート
