# MouseJumpTool

여러 모니터 간 마우스 커서를 빠르게 전환하는 Windows 도구입니다.

[English](../README.en.md) | [简体中文](README.cn.md) | [繁體中文](README.tw.md) | [日本語](README.ja.md)

## 기능

- **멀티모니터 자동 감지**: 시스템의 모든 연결된 모니터를 자동으로 감지합니다
- **타이머 표시**: 다음 모니터에 카운트다운 타이머를 표시하여 마우스를 움직이도록 유도합니다
- **커서 표시**: 둥근 직각과 원형 스타일을 모두 지원하며 현재 커서 위치를 표시합니다
- **싱글 인스턴스**: 한 번에 하나의 인스턴스만 실행됩니다
- **시스템 트레이 아이콘**: 시스템 트레이 아이콘을 통해 시작 및 종료 작업을 지원합니다

## 사용 방법

1. 프로그램을 실행하면 모든 연결된 모니터가 자동으로 감지되어 표시됩니다
2. 목표 모니터의 가장자리로 마우스를 이동하면 다음 모니터에 카운트다운이 표시됩니다
3. 카운트다운이 끝나면 커서가 목표 모니터로 이동합니다
4. 시스템 트레이 아이콘을 사용하여 프로그램을 종료합니다

### 빠른 전환

- `Ctrl + Shift + 1` 을 누르면 1 번 화면으로 전환
- `Ctrl + Shift + 2` 를 누르면 2 번 화면으로 전환
- `Ctrl + Shift + 3` 를 누르면 3 번 화면으로 전환
- 그리고 더 많은 화면으로 전환 가능

## 시스템 요구 사항

- Windows 7 또는 그 이상
- 멀티모니터 구성 지원

## 빌드

컴파일하려면 Visual Studio 에서 `MyMouneTools.sln` 솔루션 파일을 엽니다.
